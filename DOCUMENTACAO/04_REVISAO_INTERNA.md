# Controle interno de revisão

Os conteúdos foram estruturados a partir dos materiais técnicos fornecidos para Alimentação e Serviços de Saúde.

Antes da migração para o repositório oficial:

- revisar perguntas que dependem de procedimento municipal;
- revisar classificações de risco e documentação local;
- conferir vigência e texto das normas;
- validar contatos e links oficiais;
- revisar os complementos decimais antes da publicação definitiva.

Este arquivo não aparece na interface pública.
